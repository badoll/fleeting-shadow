# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-06-22

- Added the local-first memory bubble MVP.
- Added media import, random memory focus, background settings, and ambient audio.
- Added short-term Web/PWA support with manifest metadata, app-shell caching, mobile runtime tuning, and platform support documentation.
- Added media selection validation, WebGL recovery handling, and production smoke checks across desktop, mobile, Android-width, and landscape viewports.
- Added unit tests for domain, layout, and audio helper logic.
- Added open-source readiness files for public release.
