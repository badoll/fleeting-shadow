# Code Of Conduct

This project aims to be a respectful, focused place to improve a small creative browser app.

Participants are expected to:

- Be kind and constructive.
- Respect privacy, especially around personal photos and videos.
- Avoid harassment, personal attacks, or discriminatory language.
- Keep issue and pull request discussions relevant to the project.

Maintainers may edit, hide, or remove comments that make the project unsafe or unproductive.
